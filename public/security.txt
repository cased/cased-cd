# Security Policy

We take security seriously at Cased. If you discover a security vulnerability,
please report it responsibly.

## Reporting a Vulnerability

Email: security@cased.com
Subject: [SECURITY] Brief description

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

## Response Time

We aim to acknowledge reports within 48 hours and provide updates every 7 days
until the issue is resolved.

## Disclosure Policy

Please do not publicly disclose the vulnerability until we have had a chance
to address it. We will work with you to coordinate disclosure timing.

Thank you for helping keep Cased CD secure!
